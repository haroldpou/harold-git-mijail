SFAE

SFAE (Sistema de Faturamento e Análise Econômico) É um aplicativo simples e interativo suportado sobre Microsoft Excel mais muito fácil de utilizar. O sistema depois de instalado somente poderá ser executado desde esse computador. .

Cadastramento

Cadastramento inicial para um administrador e até três assistentes.

Segurança

Segurança de acesso ao Sistema através de umUtilizador e Palavra Passe

.

.

menu principal

O menu principal esta Sistema dividido em três áreas de trabalho fundamentais, facturamento, stock de armazém e a de segurança. Para o nível de assistente o Sistema só permite acessar à área de facturamento, consultar o stock mínimo e trocar de usuário conforme se requeira sem ter que abandonar o Sistema. Para o nível de administrador esta disponível o 100% das funções do sistema.

facturamento

Area de facturamento é personalizado com o logo da empresa a mesmal permite ver em tempo real o acumulado facturado, o número de facturas processadas e a classificação das mesmas em pagamento efectivo ou em pagamento eletrônico, possui uma área de consulta segundo os critérios a analisar e uma ajuda referenciada de código/produto para auxiliar o pessoal que encontra-se operando o sistema nesse momento. As facturas impressas levam o logo da empresa.

Consulta

Consulta da Base de Dados das facturas geradas.

.

.

exporta

O Sistema exporta a base de dados das facturas ou das facturas proformas geradas a uma área de trabalho onde é permitido realizar diferentes consultas como filtrado por datas, classificação de facturas segundo critérios de seleção, etc obtendo-se em tempo real o importe total para cada análise.

Cadastramento de clientes

Cadastramento de clientes permitindo alterar os dados dos mesmos sem acessar a base de dados.

Cadastramento de produtos

Cadastramento de produtos permitindo alterar os dados dos mesmos sem acessar a base de dados icluindo imagem dos mesmos.

.

.

Consulta da Base de Dados do Stock

Consulta da Base de Dados do Stock de produtos em armazém, onde o Sistema alerta sobre produtos em stock mínimo ou aqueles que encontram-se esgotados ao pessoal que esta operando o Sistema na área de faturação.

exporta relatorios

O Sistema exporta relatorios personalizados em formato PDF em função dos interesses do administrador.
